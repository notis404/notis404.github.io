With Friends

With Friends is a 3D adventure game made in Unreal Engine that I helped create for a Game Jam with the game development club at my univeristy. While this game is very simple, it was my first experience working on a game with an interdisciplinary team and my first time working with the Unreal Engine so I definitely learned a lot.

I was a beginner gameplay programmer on this project so I was responsible for implementing the ability to switch level states, UI backend, as well as some basic AI pathfinding. This was my first game project with a full team of artists and programmers so I definitely learned a lot about the development cycle for a game and how it can be optimized.
