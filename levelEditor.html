Level Editor

A level editor made in Unity 2D that I am currently developing for an unnamed 2D puzzle platforming game with Limbitless Solutions. Levels are stored and saved to a Firebase Realtime Database in the JSON file format for low overhead on storage and retrieval time. Players are authenticated via Google authentification services and levels are associated with the authenticated players who save them. In the future, players will be able to see and play other user-created levels as well.

As the solo programmer on this project, I am responsible for implementing this level editor from scratch and I also work with our animation and art teams to make sure their artwork is succesfully implemented within the level editor.

Technologies Used:

  • Unity
  • C#
  • Firebase Realtime Database
  • Google Authentification Services
  • Git
